FileLen

Returns the size of a file in bytes.

Syntax
      FileLen (file_path)

Key
   file_path   The file to inspect.


The FileLen() function can be used in VBA or in an SQL query.

Example

Dim dtmDemo As Date

dtmDemo = FileLen(F:\demo.xls)
