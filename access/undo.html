.Undo

Undo the last data edit.

Syntax
      expression.Undo

Key
   expression   The object, a form or a form control.

Example

Reset all changes to the FirstName text control:

Me!txtFirstName.undo
or
strControl = Me!txtFirstName
Me(strControl).Undo

Reset all changes to a form:

Me.Undo
