Asc

Function that returns the Ascii code which represents a specific character.

Syntax
      Asc (string)

Asc ("A") will return 65

The Asc() function can be used in VBA or in an SQL query.

Examples

In a query:

In VBA:

Dim intDemo As Integer

intDemo = Asc ("A")
> 65

intDemo = Asc ("ABC")
> 65
