Sqr

Return the square root of a number.

Syntax
      Sqr(number)

Key
   number    A valid numeric expression greater than or equal to zero.

The Sqr() function can be used in VBA or in an SQL query.

Examples

Dim intDemo as Integer
intDemo = Sqr (16)
'Returns: 4
