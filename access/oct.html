Oct

Function that converts an integer into a variant (string) containing the octal value of the number. 

Syntax
      Oct (Number)

Oct() will return numbers as large as 11 octal digits. Octal values contain the digits 0 to 7.

Example

Dim strOctal
strOctal = Oct(1234)
