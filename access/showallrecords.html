DoCmd.ShowAllRecords

Remove any applied filter from the active table, query result set, or form, and display all records.

Syntax
      Docmd.ShowAllRecords

The ShowAllRecords action doesn't have any arguments.

Example

DoCmd.ShowAllRecords
