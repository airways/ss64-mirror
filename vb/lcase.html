Lcase

Convert a string to lower-case.

Syntax
      Lcase(string)

Key
   string    A string or string expression.

Examples

Dim strDemo
strDemo = Lcase("12 ACACIA Gardens")
Msgbox strDemo
Returns "12 acacia gardens"
