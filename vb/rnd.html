Rnd

Return a pseudorandom number.

Syntax 
      Rnd[(seed)]

Key
   seed  A seed value of 0 will return the most recent random number
         A seed value > 0 will return the next random number(default)
         A seed value < 0 will return the same number

Example

  myrand = rnd()
  WScript.Echo myrand
