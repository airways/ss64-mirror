Month

Return the month (a number from 1 to 12) for a given date.

Syntax
      Month(date_value)

Key
   date_value  A valid date.

Examples

Dim intDemo
intDemo = Month(dtmShiftEnd)
intDemo = Month(#30/12/2012 08:55:00 AM#)
