Minute

Return the minute of the hour (a number from 0 to 59) given a time value.

Syntax
      Minute(time_value)

Key
   time_value  A valid time.

Examples

Dim intDemo
intDemo = minute(dtmShiftEnd)
intDemo = minute(#08:55:30 AM#)
