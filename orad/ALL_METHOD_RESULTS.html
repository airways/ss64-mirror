ALL_METHOD_RESULTS

Description of method results of types accessible to the user 

Columns
   ___________________________
 
   OWNER
      Owner of the type
   TYPE_NAME
      Name of the type
   METHOD_NAME
      Name of the method
   METHOD_NO
      Method number for distinguishing overloaded method (not to be used as ID number)
   RESULT_TYPE_MOD
      Type modifier of the result
   RESULT_TYPE_OWNER
      Owner of the type of the result
   RESULT_TYPE_NAME
      Name of the type of the result
   CHARACTER_SET_NAME
      Character set name of the result
