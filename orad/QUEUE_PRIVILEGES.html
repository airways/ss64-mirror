QUEUE_PRIVILEGES

Grants on queues for which the user is the grantor,grantee, owner or an enabled role or PUBLIC is the grantee 

Columns
   ___________________________
 
   GRANTEE
      Name of the user to whom access was granted
   OWNER
      Owner of the object
   NAME
      Name of the object
   GRANTOR
      Name of the user who performed the grant
   ENQUEUE_PRIVILEGE
      Permission to ENQUEUE to the queue
   DEQUEUE_PRIVILEGE
      Permission to DEQUEUE from the queue
