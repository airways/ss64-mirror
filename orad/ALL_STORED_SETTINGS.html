ALL_STORED_SETTINGS

Parameter settings for objects accessible to the user 

Columns
   ___________________________
 
   OWNER
      Username of the owner of the object
   OBJECT_NAME
      Name of the object
   OBJECT_ID
      Object number of the object
   OBJECT_TYPE
      Type of the object
   PARAM_NAME
      Name of the parameter
   PARAM_VALUE
      Value of the parameter
