USER_COL_PRIVS_MADE

All grants on columns of objects owned by the user 

Columns
   ___________________________
 
   GRANTEE
      Name of the user to whom access was granted
   TABLE_NAME
      Name of the object
   COLUMN_NAME
      Name of the column
   GRANTOR
      Name of the user who performed the grant
   PRIVILEGE
      Column Privilege
   GRANTABLE
      Privilege is grantable
