USER_ROLE_PRIVS

Roles granted to current user 

Columns
   ___________________________
 
   USERNAME
      User Name or PUBLIC
   GRANTED_ROLE
      Granted role name
   ADMIN_OPTION
      Grant was with the ADMIN option
   DEFAULT_ROLE
      Role is designated as a DEFAULT ROLE for the user
   OS_GRANTED
      Role is granted via the operating system (using OS_ROLES = TRUE)
