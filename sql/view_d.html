DROP VIEW

Remove one or more views from the current database. 

Syntax
      DROP VIEW [schema.] view [...,n ] [;]

Key
   view    Name of the view to be dropped.

DROP VIEW can be executed against indexed views.

Example

DROP VIEW MyView;
