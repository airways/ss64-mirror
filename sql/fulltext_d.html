DROP FULLTEXT INDEX

Remove a full-text catalog from a database. 

Syntax
      DROP FULLTEXT INDEX ON table_name

Key:
    table_name  Table containing the Full Table Index

Example

DROP FULLTEXT INDEX ON TestSchema.SalesTable;
GO
