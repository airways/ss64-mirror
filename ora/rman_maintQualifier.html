RMAN maintQualifier clause

Subclause to specify the Maintenance Schedule.

Syntax:

 { TAG [=] ['] tag_name [']
  | completedTimeSpec
  | LIKE 'string_pattern'
  | DEVICE TYPE deviceSpecifier [, deviceSpecifier]…
  | BACKED UP integer TIMES TO DEVICE TYPE deviceSpecifier
 }
