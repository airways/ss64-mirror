CREATE OUTLINE

Create a stored outline - a set of attributes used by the optimizer to generate an execution plan.

Syntax:

   CREATE [OR REPLACE] [PUBLIC|PRIVATE] OUTLINE [schema.]operator
      [FROM [PUBLIC|PRIVATE] source_outline]
         [FOR CATEGORY category] 
             ON statement;
