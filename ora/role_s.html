SET ROLE

Choose the current user role(s).

Syntax:

   SET ROLE role [IDENTIFIED BY password][,…]

   SET ROLE ALL [EXCEPT role,…]

   SET ROLE NONE

Example

SET ROLE db_user, db_superuser;
