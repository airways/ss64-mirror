CREATE SEQUENCE

Create a sequence.

Syntax:

   CREATE SEQUENCE [schema.]sequence_name option(s)

Options:
      INCREMENT BY int
      START WITH int
      MAXVALUE int | NOMAXVALUE
      MINVALUE int | NOMINVALUE
      CYCLE | NOCYCLE
      CACHE int | NOCACHE
      ORDER | NOORDER
