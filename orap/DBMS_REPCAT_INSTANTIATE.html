PL/SQL Packages DBMS_REPCAT_INSTANTIATE

Instantiate deployment templates.

Subprocedures:

DROP_SITE_INSTANTIATION  Remove the target site from the DBA_REPCAT_TEMPLATE_SITES view. 

INSTANTIATE_OFFLINE Function
                     Generate a script at the master site that is used to create the
                     mview environment at the remote mview site while offline. 

INSTANTIATE_ONLINE Function
                     Generate a script at the master site that is used to create the
                     mview environment at the remote mview site while online.

For full documentation of the packaged procedures above see the Oracle Manual:
"Oracle9i Supplied PL/SQL Packages and Types Reference"

or the book Oracle Built in Packages by Steven Feuerstein et al
