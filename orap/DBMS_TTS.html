PL/SQL Packages DBMS_TTS

Checks if a transportable set is self-contained.

Subprocedures:

TRANSPORT_SET_CHECK Check if a set of tablespaces (to be transported)
                    is self-contained. 

DOWNGRADE           Downgrade transportable tablespace related data.

For full documentation of the packaged procedures above see the Oracle Manual:
"Oracle9i Supplied PL/SQL Packages and Types Reference"

or the book Oracle Built in Packages by Steven Feuerstein et al
