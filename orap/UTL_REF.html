PL/SQL Packages UTL_REF

Reference to an Object Table.
These procedures enable you to write generic type methods without knowing the object table name. 

Subprocedures:

SELECT_OBJECT Select an object given a reference. 

LOCK_OBJECT   Lock an object given a reference. 

UPDATE_OBJECT Update an object given a reference. 

DELETE_OBJECT Delete an object given a reference.

For full documentation of the packaged procedures above see the Oracle Manual:
"Oracle9i Supplied PL/SQL Packages and Types Reference"

or the book Oracle Built in Packages by Steven Feuerstein et al
