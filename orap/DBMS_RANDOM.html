PL/SQL Packages DBMS_RANDOM

Fast random number generator - generates 8 digit integers. 

Subprocedures:

INITIALIZE  Initialize the package with a seed value. 

SEED        Reset the seed. 

RANDOM Function
            Get the random number. 

TERMINATE   Close the package.  

For full documentation of the packaged procedures above see the Oracle Manual:
"Oracle9i Supplied PL/SQL Packages and Types Reference"
