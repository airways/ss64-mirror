CLIP.exe (Resource Kit / Windows 7)

Copy the result of any command to the Windows clipboard.

Syntax
      command | CLIP

      CLIP < filename.txt

When using clip in a batch script you should warn the user that their clipboard is about to be overwritten.

For Example:

DIR | CLIP

DATE /t | CLIP
